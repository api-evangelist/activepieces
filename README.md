# Activepieces (activepieces)
Activepieces is an open-source, no-code automation platform that enables users to streamline workflows by connecting various applications and automating tasks. It supports over 200 integrations and allows developers to build custom TypeScript-based pieces. The platform offers AI agents, MCPs, and workflow automation capabilities.
